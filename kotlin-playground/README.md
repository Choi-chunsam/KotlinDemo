# Variables

val: 변경불가능한 변수, immutable

var:변경 가능한 변수

## Types

Byte, Short, Int, Long 등등 Unsigned 빼고는 다 자바랑 같다.

## non-null, nullable

Non-null

- 변수에 null 할당을 허용하지 않는 타입

var appName: String = “Shoppi”

appName = null 안됨 이거

Nullable

- 변수에 null할당을 허용하는 타입

var userName: String? = “woong”

userName = null 이거 가능
